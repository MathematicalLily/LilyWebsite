Post by Lily Dermanis, March 28th, 2019.

Below are some mathematical theorems which played a major part in computing.

This theorem refers to finding the shortest routhe via multiple locations and it is Dijkstra's algorithm. It is used in transportation and packet switching. We can interpret Dijkstra's as an array of distances from the source nodeto each node in the graph as intialised by specific variables, where the intital distance of s1 = 0 and and for the other nodes the ending distance is set to infinity until we fill them up with specific array values. This is done at the beginning because as the algorithm proceeds, the distance from the source to each node in the graph will be recalculated and finalized when the shortest distance to the destination is found. Futhermore, Dijkstra's has a big o notation of O (vElogE) so is very efficient for large sets of data and a tractable problem.
